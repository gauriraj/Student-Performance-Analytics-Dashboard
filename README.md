# 📊 Student Performance Analytics Dashboard  

> 🚀 Transforming 6,300+ student records into actionable insights for smarter academic decision-making  

---

## 🌟 Overview  
Built a **high-impact Power BI dashboard** to analyze student performance by uncovering relationships between academic outcomes and behavioral factors such as study habits, attendance, sleep patterns, and lifestyle variables.  

This project goes beyond visualization — it focuses on **identifying performance drivers, detecting risk patterns, and enabling data-driven interventions.**

---

## 🎯 Real-World Problem  
Educational institutions often struggle to answer:  

- Why are some students underperforming despite effort?  
- Which factors truly impact academic success?  
- How can at-risk students be identified early?  

This dashboard provides a **data-driven solution** to monitor, analyze, and improve student performance at scale.

---

## 🧠 Analytical Approach  

- 📊 Correlation analysis between study hours and exam scores  
- 📈 Trend analysis to track performance progression  
- 🔍 Behavioral pattern detection (sleep, attendance, engagement)  
- 🎯 Identification of performance drivers and risk indicators  
- 🧩 Multi-dimensional filtering for deeper insights  

---

## 📊 Key Dashboard Capabilities  

- 🔹 KPI Overview (Exam Score, Study Hours, Sleep, Attendance)  
- 🔹 Study vs Score Relationship Analysis  
- 🔹 Sleep Impact on Academic Performance  
- 🔹 Attendance-Based Performance Trends  
- 🔹 Performance Heatmap for pattern recognition  
- 🔹 Dynamic filtering (Gender, School Type, Internet Access, Parent Involvement)  

---

## 🔍 Key Insights  

- ✅ Strong correlation between **study hours and academic performance**  
- 📈 Higher **attendance consistently leads to better outcomes**  
- 😴 Balanced sleep contributes to stable performance trends  
- ⚠️ Low study + low attendance = **high-risk student segment**  
- 📊 Academic performance is significantly influenced by behavioral factors

- 📊 Supports **data-driven academic strategies**
